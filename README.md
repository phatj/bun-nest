## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ bun install
```

## Running the app

```bash
# development
$ bun dev

# compile
$ bun compile

# production mode
$ bun start
```

## Test

```bash
# unit tests
$ bun test

# test coverage
$ bun test --coverage
```
